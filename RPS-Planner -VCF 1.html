🔧 SDDC Toolbox

▼
Download latest SDDC ToolBox

How to run SDDC Assistant:

⚠ Take a snapshot of the SDDC Manager VM before using this tool.

1. Unzip:

root@vvxrail-sddc [ /home/vcf ]# unzip SDDCassistant_1.0.1.zip

2. Enter directory and set permissions:

cd SDDCassistant/
chmod a+x SDDCAssistant.sh

3. Run:

./SDDCAssistant.sh
Note: Please ensure you have a valid snapshot of the SDDC Manager VM before running any scripts.
